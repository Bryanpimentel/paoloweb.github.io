PLAN COMPLETO (ONLINE)

Se trata de un plan que incluye tanto la nutrición como la parte deportiva durante 90 días para que puedas comprobar unos resultados más rápidos de una forma eficaz y realista.

¿ PARA QUIÉN ESTÁ DISEÑADO ESTE PLAN?

Este plan está diseñado para todo el mundo, pero sobre todo está pensado para todos aquellos que estén empezando en este mundo o necesiten un plus de motivación. Si eres una persona que suele frustrarse con la alimentación y el deporte este es tu plan. Aquí te enseñaré a realizar correctamente las técnicas de los ejercicios y llevar una alimentación adecuada, no esperes dietas estrictas porque eso no funciona.

¿ QUE INCLUYE?

• Rutinas adaptadas a tus necesidades y objetivos.

• Recetas de los menús.

• Seguimiento semanal por videollamada o chat para poder practicar ejercicios, realizar modificaciones etc

PLAN NUTRICIONAL (ONLINE)

Un plan nutricional adaptado a tus necesidades y objetivos para que puedas cumplir de una forma realista y divertida tus objetivos. No esperes una dieta estricta ni una tortura en comer alimentos extraños y super complicados. Estoy aquí para ayudarte a cumplir tus objetivos no para complicártelos más.

¿ PARA QUIÉN ESTÁ DISEÑADO ESTE PLAN ?

Este plan está diseñado para todas aquellas personas que se vuelven locas a la hora de intentar mejorar su alimentación. No se trata de sufrir y exigir cosas innecesarias como pasar hambre, comer cosas que no te gustan o gastarte un dineral en comida todos los meses. En este plan podrás disfrutar de recetas que se adapten a tu tiempo, tus gustos y tus necesidades. Además, con el tiempo aprenderás a realizar tu rutina nutricional tú solo sin frustraciones. Solo es tiempo y constancia.

¿ QUE INCLUYE?

• Menús semanales
• Lista de la compra.
• Consejos sobre nutrición.

PLAN BASICO DE ENTRENAMIENTO

Se trata de un plan de entrenamiento diseñado para que puedas conseguir tus objetivos deportivos ( ganar masa muscular. pérdida de peso, tonificación...) de una forma objetiva , sencilla, divertida, ordenada y aprendiendo las técnicas correctas a la hora de ejecutar los ejercicios.

¿ PARA QUIÉN ESTÁ DISEÑADO ESTE PLAN ?

Este plan está diseñado para todo el mundo, pero sobre todo está pensado para todos aquellos que estén empezando en este mundo o necesiten un plus de motivación. Podrás acceder desde cualquier parte a través de mi chat donde llevaremos todo el seguimiento y procedimiento, así que no estar cerca mio no es excusa. Por ello este plan combina dos modalidades, la online y la presencial.

¿QUÉ INCLUYE?

• 3 sesiones semanales (12 sesiones mensuales) de 60 minutos aproximadamente cada una y de forma presencial.
• Rutinas de ejercicios adaptadas a tu objetivo y necesidad.
• Seguimiento diario vía chat.
• Seguimiento personalizado. Podrás comunicarte conmigo vía chat o videollamada, estar al día de las actividades gratuitas como rutas que propongo para vosotros, podrás hacer tu seguimiento alimenticio, y contestar las encuestas que os envío del seguimiento.
• ⁠Lo más importante es que podrás cumplir tus objetivos de una forma realista y organizada conmigo.
